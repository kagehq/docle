Zero-config code playground embedding with one script tag

1. Inline Code (Data Attribute)

Just add data-docle to any element:

print("Hello from Docle!") print("This is inline code") for i in range(3): print(f"Count: {i}") 
<script src="https://docle.workers.dev/embed.js"></script>

<div data-docle data-lang="python">
print("Hello from Docle!")
</div>

2. Node.js Example

console.log("Running Node.js!"); console.log("Math.PI =", Math.PI); console.log("Current time:", new Date().toISOString()); 
<div data-docle data-lang="node">
console.log("Hello, Node!");
</div>

3. Light Theme

def greet(name): return f"Hello, {name}!" print(greet("World")) 
<div data-docle data-lang="python" data-theme="light">
...
</div>

4. Custom Height

# Shorter playground print("Compact view") 
<div data-docle data-height="300px">
print("Compact!")
</div>

5. Code via Attribute

Use data-code instead of innerHTML:

<div data-docle 
     data-lang="python" 
     data-code="import random\nprint(random.randint(1, 100))">
</div>

6. Programmatic Control

Use the JavaScript API for advanced control:

x = 42 print(f"The answer is {x}") 
// Access the embed instance
const element = document.getElementById('controlled');
const embed = new DocleEmbed(element);

// Run code
embed.run();

// Update code
embed.setCode('print("New code!")');

That's it! One script tag, zero configuration.
